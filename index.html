URL Shortener

Specification

URL Shortener project completed as part of FreeCodeCamp curriculum.

API accepts any valid URL and provides a shortener version in JSON that can be used to redirect

Providing already shortener URL will redirect to the original URL

Project built using:

  • Vanilla JS
  • Node
  • Express

Example input:

https://little-url.herokuapp.com/new/https://www.google.com
https://little-url.herokuapp.com/new/https://www.google.com

Example output:

{ "original_url":"http://foo.com:80", "short_url":"https://little-url.herokuapp.com/8170" }

Usage

https://little-url.herokuapp.com/new/https://www.google.com

Redirects to:

https://little-url.herokuapp.com/new/https://www.google.com
